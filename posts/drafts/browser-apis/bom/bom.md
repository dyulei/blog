BOM 浏览器对象模型
==============

`BOM` （Browser Object Model，浏览器对象模型）提供了通过 `JavaScript` 访问和控制浏览器窗口（window）、显示器（screen）与浏览历史（history）等对象的接口。

`BOM` 并非 W3C 的标准，而是在各家浏览器中逐渐形成的一致性接口。

`BOM` 可以看做是 Browser API 的一部分。

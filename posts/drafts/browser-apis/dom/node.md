Node 对象
========

`Node` 对象代表了 DOM 中的节点，也是所有 DOM 元素的父类。

属性
----

###nodeName

节点名称。

节点名称的规则：

+ 元素节点以大写的标签名作为节点名称
+ 属性节点以属性名作为节点名称
+ 文本节点以 #text 作为节点名称
+ 文档节点以 #document 作为节点名称

###nodeValue

节点的值。

节点值得规则：

+ 文本节点的值为节点包含的文本
+ 属性节点的值为属性值
+ 文档节点和元素节点没有 nodeValue 属性（undefined）

###nodeType

节点类型。

节点类型列表：

+ 1 元素节点
+ 2 属性节点
+ 3 文本节点
+ 8 注释节点
+ 9 文档节点

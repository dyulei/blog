溢出（overflow）
==============

overflow
--------

设置内容溢出时，元素的行为。

属性值：

1. visible 溢出部分可见（默认）
2. hidden 溢出部分不可见
3. scroll 元素出现滚动条（不论内容是否溢出）
4. auto 内容溢出时，才出现滚动条

overflow-x
----------

设置内容溢出时，元素在横向的行为。

属性值：

1. visible
2. hidden
3. scroll
4. auto
5. no-display 内容溢出时，隐藏整个元素
6. no-content 内容溢出时，隐藏所有内容

overflow-x
----------

设置内容溢出时，元素在纵向的行为。

overflow-style
--------------

设置内容溢出时，元素的滚动方式。

属性值：

1. auto
2. scrollbar 滚动条
3. panner
4. move 内容可拖动
5. marquee 内容自动滚动

外边距(margin)
=============

margin-top
----------

设置顶部外边距。

属性值：

1. auto 自动
2. 尺寸值
3. 百分值

margin-right
------------

设置右侧外边距。

margin-bottom
-------------

设置底部外边距。

margin-left
-----------

设置左侧外边距。

margin
------

在同一个声明中设置内边距属性。

可包括的属性：

+ margin-top
+ margin-right
+ margin-bottom
+ margin-left

有四种设置方式：

1. a b c d 分别设置顶部、右侧、底部和左侧的外边距
2. a b c 分别设置顶部、左右和底部的外边距
3. a b 分别设置顶部和底部、左右的外边距
4. a 一次设置四个外边距
